Playwright.io Typescript Framework including

- Examples of basic playwright tests for UI and API
- eslint configuration
- Github Actions for linting, test execution and CodeQL analysis

## Installation

```bash
npm install
npx playwright install
```

### Run UI tests

```bash
npm run tests:ui
```

### Run API tests

```bash
npm run tests:api
```
